# Coverage

This MCP provides access to official Czech data protection regulatory data from **ÚOOÚ** (Úřad pro ochranu osobních údajů — Czech Data Protection Authority).

## Scope

| Category | Content |
|----------|---------|
| **Decisions** | GDPR enforcement decisions, sanctions, and reprimands |
| **Guidelines** | Official guidance documents, opinions, recommendations, and FAQs |
| **Topics** | Controlled vocabulary with Czech and English topic names |

## Topics Covered

- Consent (souhlas)
- Cookies
- International data transfers (mezinárodní předávání)
- Data Protection Impact Assessment / DPIA
- Breach notification (ohlašování porušení)
- Privacy by design
- CCTV / camera systems (kamerové systémy)
- Health data (zdravotní údaje)
- Children's data (data dětí)

## Data Source

| Field | Value |
|-------|-------|
| **Authority** | ÚOOÚ — Úřad pro ochranu osobních údajů |
| **URL** | https://www.uoou.cz/ |
| **Language** | Czech (cs) |
| **Update frequency** | Periodic |
| **License** | Public regulatory data — official government publications |

## Limitations

- Coverage may be incomplete relative to the full ÚOOÚ publication archive
- Database updates are periodic and may lag official publications
- This is a research tool, not a substitute for professional regulatory counsel
- Always verify references against primary sources at https://www.uoou.cz/

## Freshness

Use `cz_dp_check_data_freshness` to check the latest record dates and record counts in the live database.
Use `cz_dp_list_sources` for full source provenance metadata.
